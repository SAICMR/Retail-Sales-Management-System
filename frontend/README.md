# Retail Sales Management System - Frontend

## Overview
React-based frontend application for the Retail Sales Management System, providing an intuitive interface for searching, filtering, sorting, and paginating sales data.

## Tech Stack
- React 18
- Vite
- Axios

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The application will run on `http://localhost:3000`

3. Build for production:
```bash
npm run build
```

## Features

- **Search**: Real-time search across customer names and phone numbers
- **Filters**: Multi-select filters for regions, gender, categories, tags, payment methods, and range filters for age and dates
- **Sorting**: Sort by date, quantity, or customer name with ascending/descending options
- **Pagination**: Navigate through results with 10 items per page
